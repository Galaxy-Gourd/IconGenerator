# Changelog

## [1.0.0]
- Initial release.
- Transparent prefab icon capture for URP via two-pass black/white alpha reconstruction.
- Grid-cell (tetris) and square output resolutions with supersampling.
- Orthographic/perspective framing with adjustable margin.
- Orientation presets + custom Euler.
- Configurable key/fill/rim lighting rig with presets and flat ambient.
- Live in-window preview sharing the capture render path.
- Single + batch capture (multi-selection, recursive folder selection, and folder picker), auto Sprite import, savable/loadable preset assets.
